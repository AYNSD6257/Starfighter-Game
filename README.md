# Starfighter-Game
This was a high school project where we were tasked with creating and designing our own game using Java codes and complex libraries. This is basically a replica of the famous arcade Galactic Shooter game. All files will be linked for your perusal. A Java-based arcade shooter game developed using object-oriented programming principles.

## Features
- Player-controlled spaceship
- Alien enemy waves
- Projectile shooting system
- Collision detection
- Keyboard controls
- Custom graphics

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Swing / Graphics

## Learning Outcomes
This project strengthened my understanding of:
- Classes and objects
- Inheritance
- Collections
- Event handling
- Game development fundamentals
